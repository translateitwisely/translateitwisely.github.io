Recursos

Software útil para traductores:

TRADOS

Herramienta para traducir más rápido. Permite editar, revisar y gestionar proyectos de traducción, así como terminología corporativa.

OMEGA T

Alternativa a SDL Trados Studio o Wordfast. Permite utilizar memorias de traducción, glosarios externos, corrector ortográfico.

PDF- XChange Viewer

Lector de PDF gratuito. Permite hacer comentarios, resaltar texto, colocar sellos y muchas otras cosas más. Bastante útil para la revisión y corrector de textos.

ApSic XBench

Herramienta para el control de calidad. Permite revisar cuestiones básicas de control de calidad como dobles espacios introducidos erróneamente, oraciones que hayan quedado sin traducir, oraciones fuente que se hayan traducido de diferente manera, etc.

PhraseExpress

Software que permite generar textos largos que permite reproducirlos después con una simple combinación de caracteres.

Cursos de perfeccionamiento o especialización:

Corrección de cualquier tipo de documento: manuales, novelas, artículos etc. Revisión de traducciones automáticas, humanas y automatizadas. Corrección ortotipográfica con una excelente precisión y dedicación.

Recursos

Software útil para traductores:

TRADOS

Tool that allows the user to translate more quickly. It allows editing, revising and managing projects of translations as well as corporative terminology.

OMEGA T

Alternative to SDL Trados Studio or Wordfast. It allows the user to use translation memories, external diccionaties and spelling checker.

PDF- XChange Viewer

Free PDF reader. It allows the user to make some comments, highlight texts, affix stamps and other useful things. It is very useful for the review and correction of texts.

ApSic XBench

Tool to control the quality of the translation. It allows the user reviewing basic issues to control the quality such as double-spaced wronly inserted, sentences that have not been translated by a mistake, sentences that have to be translated with idioms.

PhraseExpress

Software that allows the user to generate long texts and reproduce them after a simple combination of characters.
